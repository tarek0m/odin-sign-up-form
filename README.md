# Sign-Up Form

This project is an assignment from [The Odin Project](https://www.theodinproject.com/lessons/node-path-intermediate-html-and-css-sign-up-form) to practice creating an HTML form. The form is styled using CSS and is hosted using GitHub Pages.

## Live Demo

You can view the live demo of the project [here](https://tarek0m.github.io/odin-sign-up-form).

## Project Description

The goal of this project is to create a sign-up form that collects user information. The form includes fields for the user's name, email, password, and other relevant details. The form is styled to be visually appealing and user-friendly.

## Features

- Custom CSS styling
- HTML Form validation

## Technologies Used

- HTML
- CSS

## Installation

To run this project locally, follow these steps:

1. Clone the repository:

```bash
git clone https://github.com/tarek0m/odin-sign-up-form.git
```

2. Navigate to the project directory:

```bash
cd odin-sign-up-form
```

3. Open `index.html` in your browser to view the form.
